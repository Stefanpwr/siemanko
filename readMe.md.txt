Short Tutorial how to run project.
First of all You have to install node.js and npm in your pc. Then add npm to your Windows* PATH.
Now, cd to a place when u have ur project, and cd to directory nodelogin.
Now run XAMPP with MySQL and type node login.js in your terminal to run the server.
Open your browser and type : localhost:3000 (You can change the port at the end of login.js file (serve file) by changing app.listen(3000)
to another port.
